# zapret-apk
zapret android edition
A DPI bypass VPN client that uses `zapret` to evade censorship.  
Built by kl3vr – original zapret by bol-van.

## ⚠️ Requirements
- **Root access** (mandatory because zapret manipulates raw packets)
- **zapret binary** (ARM64/ARMv7) placed in `/data/data/com.kl3vr.zapretvpn/files/zapret`
- Android 8.0+
